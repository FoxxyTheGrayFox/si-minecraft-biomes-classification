Dataset jest dość mały bo nie chciało mi się screenshotów robić i rozpoznaje tylko idealnie wybrane biomy bez struktur, shaderów itd. 8 biomów tylko jest bo Minecraft ma kilka bardzo podobnych biomów, które byłyby trudne do klasyfikacji (a dwa że w grze jest jakieś 67 biomów życia by mi zabrakło). Tak więc dataset obejmuje tylko 8 biomów, które są dość charakterystyczne:
1. Basalt Deltas (nether)
2. Cherry Grove (overworld)
3. Crimson Forest (nether)
4. End Highlands (end)
5. Forest (overworld)
6. Lush Cave (overworld)
7. Ocean (overworld)
8. The End (end)
